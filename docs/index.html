Rocqman

A Pacman-like arcade game written in Rocq, extracted to C++ with Crane, and rendered in the browser through SDL2 and WebAssembly.

Rocq Crane C++ extraction SDL2 Emscripten WebAssembly

Gameplay

Clear the maze by eating every dot and power pellet. Power pellets frighten the ghosts for a short time, letting Rocqman eat them for bonus points.

  • Dots are worth 10 points.
  • Power pellets are worth 50 points.
  • Frightened ghosts are worth 200 points.
  • Touching a chasing ghost costs one life.

Controls

  • Use arrow keys or WASD to buffer movement.
  • Space pauses and unpauses.
  • Q or Esc quits the current run.

Build Notes

The native project builds with make. This page is generated by make web, which preloads the assets directory for the sprite and sound effects.
